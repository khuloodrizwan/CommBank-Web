Step 1 — Create a new branch called icons:
```
bashgit checkout -b icons
```
Step 2 — Stage and commit all your changes:
```
bashgit add .
git commit -m "Support icons"
```
Step 3 — Push the branch to GitHub:
```
bashgit push -u origin icons
```
Step 4 — Go to GitHub:

Open your repository on GitHub
You will see a prompt to create a Pull Request for the icons branch
Click "Compare & pull request"
Add a title and description
Click "Create pull request"
